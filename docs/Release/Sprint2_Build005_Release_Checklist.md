# Sprint 2 Build005 Release Checklist

## Acceptance criteria

- [ ] Wrapper is additive.
- [ ] Legacy script is launched as a subprocess.
- [ ] Wrapper output is outside the legacy tree.
- [ ] Stdout, stderr, return code, duration, and metadata are captured.
- [ ] Legacy script SHA-256 is recorded.
- [ ] Dry-run mode succeeds.
- [ ] Timeout behavior is controlled.
- [ ] Missing legacy files produce a controlled error.
- [ ] Full test suite passes.
- [ ] Ruff passes.
- [ ] Coverage remains at or above 80%.
- [ ] Legacy preservation tests pass.
- [ ] No pre-existing legacy file is modified.
- [ ] GitHub Actions passes.
- [ ] Pull request targets `develop`.
