## NRHIS Sprint 2 — Build049

Build049 adds repository-location safety for Windows environments where OneDrive can lock or attempt to synchronize Git internal files.

### Included

- OneDrive path detection
- Guided automatic migration to `C:\GitHub\NRHIS`
- Exact `develop` synchronization without pruning
- Verified transfer of the root ZIP and checksum
- External migration receipt
- Automatic continuation of the existing one-step lifecycle

This is a pre-release and must not be marked as latest.
