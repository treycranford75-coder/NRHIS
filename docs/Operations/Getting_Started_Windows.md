# Getting Started on Windows

1. Extract the NRHIS ZIP into `Documents\\GitHub\\NRHIS`.
2. Open PowerShell in that folder.
3. Run `Set-ExecutionPolicy -Scope Process Bypass`.
4. Run `.\\scripts\\Initialize-NRHIS.ps1`.
5. Run `.\\scripts\\Migrate-Legacy-Pass1.ps1`.
6. Review `git status` before the first commit.
