## NRHIS Sprint 2 — Build050

Build050 introduces the production USGS Nueces Basin Harvest Engine.

The build can retrieve current observations for Cotulla, Tilden, Three Rivers, Mathis, Bluntzer, and Calallen; preserve the raw API response; normalize available discharge, stage, temperature, and conductance readings; flag stale, provisional, and missing data; append new observations without duplication; and publish operational JSON/CSV products with a machine-readable receipt.

This is a pre-release and must not be marked as latest.
